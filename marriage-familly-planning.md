## 结婚登记人数

```解释
  -  指某地区报告期内，符合《婚姻法》要求，在该地民政部门登记结婚的全部人数。取自民政部门统计年报。
```

## 结婚登记件数

```解释
  -  指报告期内经民政部门婚姻登记机关审查，确认当事人符合结婚条件，批准登记结婚的案件总数。取自民政部门统计年报。
```

## 涉外及华侨、港澳台居民登记结婚数

```解释
  -  指在县级（或县级以上）民政部门办理的，夫妻双方或一方是外国人、华侨、港澳台同胞的婚姻案件总数。取自民政部门统计年报。
```
